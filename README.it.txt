Dai a Homey l'opportunità di essere divertente creando cene casuali di assurdità

Non aspettarti di ricevere una cena che puoi cucinare o mangiare ;)

Aggiungi le tue parole

- Apri le impostazioni (configura l'app)
  - Scorri verso il basso fino a 'Parole personalizzate'
  - Compila le parole nelle sezioni desiderate

Utilizzo

- Crea un flusso
  - Aggiungi la scheda azione 'Crea cena casuale' e personalizzala a tuo piacimento
  - Aggiungi una scheda azione per mostrare o dire la cena casuale creata usando il tag 'Cena casuale'
    - Aggiungi un ritardo di 1 secondo alla scheda azione per consentire il completamento della prima scheda azione

Ge Homey chansen att vara rolig genom att skapa slumpmässiga nonsensmiddagar

Förvänta dig inte att få en middag som du kan laga eller äta ;)

Lägg till dina egna ord

- Öppna inställningar (konfigurera appen)
  - Scrolla ner till 'Egna ord'
  - Fyll i orden i de sektioner du vill

Användning

- Skapa ett flöde
  - Lägg till åtgärdskortet 'Skapa slumpmiddag' och justera det efter din smak
  - Lägg till ett åtgärdskort för att visa eller säga den skapade slumpmässiga middagen med taggen 'Slumpmiddag
    - Lägg till en 1 sekunders fördröjning på åtgärdskortet för att låta det första åtgärdskortet bli klart
